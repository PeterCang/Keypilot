# Keypilot

Keypilot is a cross-platform desktop tool for managing and switching API keys used by AI development tools.

## Purpose

Keypilot focuses on local, fast API key management for tools such as Claude Code, OpenAI Codex CLI/App, and Gemini CLI. It is planned as a Tauri desktop application with a system tray workflow for quick key switching.

## Planned Tech Stack

- Tauri
- Rust backend
- React + TypeScript frontend
- Local JSON storage
- Windows and macOS support

## Core Features

- Manage multiple API keys locally
- Switch active keys for supported AI development tools
- Support Claude Code, Codex CLI/App, and Gemini CLI
- Provide a system tray quick-switch workflow
- Detect and install supported AI development tools
- Store data locally in a readable JSON file
